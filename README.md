# Описание задачи

Добавить функционал к теме, где при инициализации будет создаваться кастомный тип постов `Employees`.  
Для админа на странице создания/редактирования поста должно выводиться 2 произвольных поля - `Name` и `Salary`.  
Также создать шорткод, при использовании которого будет осуществлен вывод значения из поля `Name` и значения из поля `Salary` всех постов типа `Employees`.  

Параметры шорткода:

* `salary_min` = минимальное значение поля `Salary`, при котором пост попадает в цикл;
* `salary_max` = максимальное значение;
* `sort` = ASC|DESC - сортировка вывода постов по значению поля `Salary`  

При неуказанном параметре посты сортируются по дате добавления.  
Пример  

```
[Employees salary_min=“100" salary_max=“1000” sort=“ASC”]
```
